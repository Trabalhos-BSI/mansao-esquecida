---
## `commands`
Responsável: Guilherme

O pacote de comandos é responsável por desenvolver os comandos disponíveis.

## `entities`
Responsável: Athos

O pacote de entidades é responsável por desenvolver os fantasmas e o jogador.

## `items`
Responsável: Enzo

O pacote de items é responsável por desenvolver os items disponíveis e seus usos.

## `locations`
Responsável: Davi

O pacote de locais é responsável por desenvolver o mapa, sendo a mansão e as salas.

## `dispute`
Responsável: Cleber

O pacote de disputas é responsável por desenvolver os cenários de disputa 
com os fantasmas.

## `Game`
Responsável: Jonathan

O Game tem a função de integrar todas os demais pacotes e classes.
